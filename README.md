📘 HTML_HCS – HTML Learning Project
📌 About This Project

I created this project only for learning and revising HTML basics.
This project helped me understand how different HTML tags work and how a simple website is structured using only HTML.

There is no CSS or JavaScript used in this project.
Everything is built using pure HTML to strengthen my fundamentals.

🎯 Purpose of This Project

To practice HTML from scratch

To revise important HTML topics

To understand how sections and navigation work

To build a basic registration form

To improve my basic frontend knowledge

📂 Project Structure

This project contains:

Index.html – Main tutorial page

Form.html – Complete registration form

success.html – Submission success page

Image file – Used inside the tutorial

📖 Topics Covered in This Project

The main tutorial page includes short explanations and examples of:

HTML Introduction

Basic HTML Structure

HTML Elements

HTML Attributes

Headings (h1 to h6)

Paragraph

Inline Style

Text Formatting (bold, italic, strong, em)

Quotation and Citation

Links (with target="_blank")

Images

Div

Buttons

Ordered List

Unordered List

Each topic is placed inside a separate section for better understanding.

📝 Registration Form Features

The form includes:

Text input (Full Name)

Email input

Password field

Number input

Date input

Radio buttons (Gender selection)

Checkboxes (Skills selection)

Dropdown (Country selection)

Textarea (Address)

File upload option

Reset button

Submit button

The form uses the GET method and redirects to a success page after submission.

✅ Success Page

After submitting the form, the user is redirected to a simple success page that shows:

A green success icon

"Submitted Successfully!" message
