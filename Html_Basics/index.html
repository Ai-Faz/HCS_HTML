HTML Introduction

HTML stands for HyperText Markup Language. It is used to create web pages.

HTML Basics

Basic Structure of HTML:

<html>

</head>
<body>
Content Here
</body>
</html>

HTML Elements

An HTML element consists of a start tag, content, and an end tag.
